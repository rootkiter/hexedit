# Install

## Blank Environment

```
    $ git clone https://github.com/rootkiter/vim-hexedit.git ~/.vim
```

## Under Pathogen

```
    $ git clone https://github.com/rootkiter/vim-hexedit.git ~/.vim/bundle/vim-hexedit
```
